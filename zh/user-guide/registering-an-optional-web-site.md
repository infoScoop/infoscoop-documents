# 登记任意的网站

可以作为迷你浏览器为个性化领域追加任意的网站。
打开[内容追加]标签，在文本框中输入网站的URL，单击[预览]按钮。

![Add Contents]

单击预览下的[追加]按钮，所登录的网站的迷你浏览器被显示。

![Added MiniBrowser Gadget]

**注意** URL 用的半角英数字到 2000个文字能输入可能。如果如果包含全角字符，全角字符作为约9个文字计算。
