# 访问统计

点击RSS阅读器，通知的菜单里的(![Access statistics icon]访问统计)会显示对应RSS的访问统计。

![The screen of the access statistics]

通知，RSS阅读器(也包含复合RSS阅读器)进行以下的操作的时候，在访问统计中被使用的信息会被发送到infoScoop服务器。

* 点击RSS项的链接
* 展开RSS项的详细信息
* 最大化的状态下阅览RSS的项目
