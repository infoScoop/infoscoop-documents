# 关于InfoScoop在复数个浏览器中被同时启动

如果在别的浏览器中同时启动infoScoop的情况下，最后启动的infoScoop可以利用全部的功能，在另外的浏览器中的infoScoop则自动地改换为参照模式。

参照模式下更新了自定义信息后，在首部以下如下图一样的信息将被显示。

![The message when you start the infoScoop in multiple browsers]

这时，自定义信息的更新被废弃。
