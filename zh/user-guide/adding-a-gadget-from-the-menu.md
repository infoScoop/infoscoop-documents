# 从顶部菜单追加小工具

说明一下从顶部菜单对个性化领域追加小工具的操作方法。

![Dropping the gadget from the menu]

1. 开始拖菜单项目的RSS图标。
2. 放到喜欢的地方。
3. 登记在所选菜单项目里的小工具会被显示。

这个例子是归总追加复合RSS阅读器。
小工具的配置状态自动地被服务器保存。

## 一并向下拖放

在同阶层的RSS阅读器作为复合RSS阅读器被归总追加时要进行以下的操作。

![Dropping all gadgets from the menu]

1. 拖[一并向下拖放]标签。
2. 放到喜欢的地方。
3. 同阶层的RSS阅读器及 复合RSS阅读器的图标的菜单被归总，复合RSS阅读器被做成。
