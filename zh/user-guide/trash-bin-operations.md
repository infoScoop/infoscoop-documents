# 回收站的操作

从用户面板删除的小工具可以从回收站复原。
点击命令栏的回收站图标（ ![Trash icon] ）。

![Trash Box]

<table>
    <thead>
        <tr>
            <th>号码</th><th>说明</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>(1)</td>
            <td>
                右键单击所需的项目时，菜单将显示。
            </td>
        </tr>
        <tr>
            <td>(2)</td>
            <td>
                选择的小工具
                <ul>
                    <li>还原: 还原从回收站删除的项目到个性化领域。</li>
                    <li>
                        删除: 从回收站中删除项目。<br>
                        <strong>注意</strong> 删除的项目不能被还原。
                    </li>
                    <li>取消: 取消项目的编辑。</li>
                </ul>
            </td>
        </tr>
        <tr>
            <td>(3)</td>
            <td>
                清空回收站。<br>
                不能还原删除掉的项目
            </td>
        </tr>
        <tr>
            <td>(4)</td>
            <td>
                关闭回收站。
            </td>
        </tr>
    </tbody>
</table>
